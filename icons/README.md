# أيقونات التطبيق (PWA)

لتفعيل "إضافة إلى الشاشة الرئيسية" بشكل كامل، أضِف الأيقونات التالية في هذا المجلد:

| الملف         | الحجم    | الاستخدام                    |
|---------------|----------|------------------------------|
| `icon-192.png` | 192×192  | أيقونة الشاشة الرئيسية      |
| `icon-512.png` | 512×512  | شاشة التحميل والأجهزة الكبيرة |

- الصيغة: PNG بخلفية شفافة أو بلون الخلفية `#f8f9fa`.
- يمكن استخدام نفس التصميم بحجمين مختلفين.

بدون هذه الملفات، المتصفح قد يستخدم لقطة شاشة أو أيقونة افتراضية.
